## Endpoints

### GET /nid

Fetches information for all users in the National ID Database.

- Example: `GET /nid`
- Response Format:

  ```json
  [
    {
      "id": "123",
      "first_name": "John",
      "last_name": "Doe",
      "dob": "1990-05-20",
      ...
    },
    {
      "id": "124",
      "first_name": "Jane",
      "last_name": "Smith",
      "dob": "1988-12-15",
      ...
    },
    ...
  ]
  ```

### POST /nid

Adds a new user to the National ID Database.

Example: POST /nid
Request Format:

```json
{
"first_name": "New",
"last_name": "User",
"dob": "1995-08-10",
...
}
```

Response Format:

```json
{
"id": "125",
"first_name": "New",
"last_name": "User",
"dob": "1995-08-10",
...
}
```

### GET /nid/:id

Retrieves specific user information from the National ID Database based on the provided ID.

Example: GET /nid/123
Parameters: id (user's unique identifier)
Response Format:

```json
Copy code
{
"id": "123",
"first_name": "John",
"last_name": "Doe",
"dob": "1990-05-20",
...
}
```

### PUT /nid/:id

Updates user information in the National ID Database based on the provided ID.

Example: PUT /nid/123
Parameters: id (user's unique identifier)
Request Format:

```json
{
"first_name": "Updated",
"last_name": "User",
"dob": "1990-05-20",
...
}
```

Response Format:

```json
{
"id": "123",
"first_name": "Updated",
"last_name": "User",
"dob": "1990-05-20",
...
}
```
